## La Visión

LeDesign es una plataforma integral de diseño ingenieril para Chile. Tres módulos core (Estructural, Pavimentos, Hidráulico) con potencial de expansión a todo Chile y eventualmente Latinoamérica.

**El objetivo:** Crear la herramienta estándar que usan los ingenieros civiles chilenos para diseñar, cumpliendo normativa NCh automáticamente.
